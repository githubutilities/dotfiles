# Ubuntu branch of Mathias’s dotfiles

## Installation

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/githubutilities/dotfiles/ubuntu/.ubuntu/bootstrap.sh)"
```

## Develop Environment Install Scripts

Navigate to `./.ubuntu` and pick the develop environment to install, e.g. you can install `linuxbrew` through `./.ubuntu/linuxbrew/install.sh`.
